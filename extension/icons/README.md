# Extension Icons

This directory should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create simple icons or use placeholder images. The icons are referenced in `manifest.json`.

For a quick start, you can:
1. Create simple colored squares with the text "KP" (Kalshi Pulse)
2. Use an icon generator tool
3. Use placeholder images from any icon library

The extension will work without icons, but Chrome will show a default icon instead.
